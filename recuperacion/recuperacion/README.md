# Recuperación - Algoritmo de Dijkstra

Este proyecto implementa el algoritmo de Dijkstra usando listas de adyacencia en Java.

## Grafo
- Cali → Pereira (105)
- Pereira → Bogotá (130)
- Bogotá → Valledupar (390)
- Valledupar → Barranquilla (100)
- Barranquilla → Cartagena (200)

## Cómo ejecutar
1. Compilar:
```bash
javac src/*.java
```
2. Ejecutar:
```bash
java src.Main
```

## Salida
- Distancias desde Cali.
- Camino más corto a Cartagena.
- Camino más largo desde Cali.
