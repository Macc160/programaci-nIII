# Recuperación - Algoritmo de Dijkstra

Este proyecto implementa el algoritmo de Dijkstra usando listas de adyacencia en Java.

## Instrucciones del profesor
1. Implementar Dijkstra con listas de adyacencia.
2. Usar el mapa de Colombia (Cali como nodo inicial, Cartagena como nodo final).
3. Mostrar el camino más corto y el más largo.

## Grafo
- Cali → Pereira (105)
- Pereira → Bogotá (130)
- Bogotá → Valledupar (390)
- Valledupar → Barranquilla (100)
- Barranquilla → Cartagena (200)

## Ejecución
Compilar:
```bash
javac src/*.java
```
Ejecutar:
```bash
java src.Main
```

## Salida esperada
- Distancias desde Cali.
- Camino más corto a Cartagena.
- Camino más largo desde Cali.
