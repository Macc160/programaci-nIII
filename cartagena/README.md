# Proyecto AF3 - Dijkstra + Métodos de Ordenamiento

Este proyecto implementa:
- Algoritmo de **Dijkstra** con listas de adyacencia para calcular la ruta más corta entre ciudades de Colombia (Cali → Cartagena).
- Métodos de ordenamiento: Burbuja, Burbuja Mejorado, QuickSort, MergeSort, InsertionSort, SelectionSort.
- Estructuras adicionales: Árbol Binario, Cola, Ciclos.

## Cómo ejecutar
1. Compilar:
```bash
javac *.java
```
2. Ejecutar:
```bash
java Main
```

## Salida esperada
- Distancias desde Cali a todas las ciudades.
- Ruta más corta a Cartagena.
