# AF3 - PF - Dijkstra + Métodos de ordenamiento

Proyecto entregable para la AF3: Implementación de Dijkstra y métodos de ordenamiento.

## Estructura del proyecto
```
AF3_Dijkstra_Ordenamiento/
├── src/
│   ├── Nodo.java
│   ├── Dijkstra.java
│   ├── BubbleSort.java
│   ├── BubbleMejorado.java
│   ├── SelectionSort.java
│   ├── InsertionSort.java
│   ├── QuickSort.java
│   ├── MergeSort.java
│   ├── ArbolBusqueda.java
│   └── Ciclos.java
└── README.md
```

## Compilar y ejecutar
Desde la carpeta raíz del proyecto:
```bash
javac src/*.java
java -cp src Ciclos
```

El programa muestra un menú para probar los ordenamientos, Dijkstra y análisis del árbol.
