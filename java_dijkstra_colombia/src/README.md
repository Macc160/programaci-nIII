# Proyecto AF3 - Algoritmo de Dijkstra con Ordenamiento Burbuja

## Descripción
Este proyecto implementa el algoritmo de **Dijkstra** utilizando **listas de adyacencia** en Java para encontrar la ruta más corta entre ciudades de Colombia, desde **Cali** hasta **Cartagena**. Además, se aplica el método de ordenamiento **Burbuja** para organizar los recorridos por distancia.

## Ciudades en el grafo
- Cali
- Pereira
- Medellín
- Bogotá
- Barranquilla
- Cartagena
- Valledupar

## Funcionalidades
- Implementación del grafo con listas de adyacencia.
- Algoritmo de Dijkstra para calcular distancias mínimas.
- Método Burbuja para ordenar las distancias.

## Cómo ejecutar
1. Compila el proyecto:
   ```bash
   javac src/*.java
   ```
2. Ejecuta el programa:
   ```bash
   java src.Main
   ```

## Salida esperada
- Lista de distancias desde Cali a todas las ciudades.
- Ruta más corta a Cartagena.
