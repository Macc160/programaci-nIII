# Proyecto: Algoritmo de Dijkstra en Java con Grafo de Colombia

Este proyecto implementa el algoritmo de Dijkstra usando listas de adyacencia en Java para encontrar el camino más corto entre Cali y Cartagena, basado en el grafo proporcionado.

## Instrucciones:
1. Compila el proyecto:
   ```bash
   javac src/*.java
   ```
2. Ejecuta el programa:
   ```bash
   java -cp src Main
   ```
3. El grafo está basado en el mapa de la imagen con distancias aproximadas.

## Archivos:
- src/Grafo.java: Clase para manejar el grafo.
- src/Dijkstra.java: Implementación del algoritmo.
- src/Main.java: Programa principal.
- grafo.png: Diagrama del grafo.
- ejemplos/: Ejemplos adicionales.
