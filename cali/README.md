# Proyecto: Algoritmo de Dijkstra con Grafo de Colombia

Este proyecto implementa el algoritmo de Dijkstra usando listas de adyacencia para encontrar el camino más corto entre Cali y Cartagena, basado en el grafo proporcionado.

## Instrucciones:
1. Ejecuta `main.py` para ver el recorrido más corto entre Cali y Cartagena.
2. El grafo está basado en el mapa de la imagen con distancias aproximadas.
3. Código completamente documentado para explicación en video.

## Archivos:
- dijkstra.py: Implementación del algoritmo.
- ordenamiento.py: Ordena recorridos.
- main.py: Ejecuta todo.
- grafo.png: Diagrama del grafo.
- ejemplos/: Ejemplos adicionales.
